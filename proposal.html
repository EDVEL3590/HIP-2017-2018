Proposal

1. Project Title
a. Object Mapping via Time-Distance Correlations
2. High Level Project Description
a. Utilize sound sensors to correlate time to distance.
b. Utilize time to distance correlation to navigate a robot.
c. The question: Is echolocation a viable form of detection?
3. Goals of your project
a. Discover which sound sensor would be most effective.
b. Utilize the speed of sound to test if the time-distance correlation is correct.
c. Build a robot that can utilize time-distance correlation to navigate. Utilize the information from the sensors to create a python-based code.
d. Integrate the sensors onto the robot.
4. Way to scale back project
a. Get rid of the robot aspect.
b. Instead of a robot, create a sensor analysis.
5. Reach Goal
a. Add more sound sensors to find out the best placement
b. Utilize another type of sensor
c. Test to check if the ultrasonic sensor is more effective than other sensors
d. Ultimate Reach Goal: Map out a room
6. Risks
a. Not being able to integrate the sensors onto the robot
b. Incorrect correlation of time and distance
c. Unable to program with Python
7. Conclusion
a. I am excited to learn how to program something from the beginning and learn new programming overall. I am also excited to learn how to utilize Raspberry Pi to achieve these goals.
b. This topic excites me because I am very interested in sound. I want to be able to create a program that takes it into account and then integrate the program into a robot. I am excited to see if I am up to the task of creating something with the proper materials.
